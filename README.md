# Sales & Revenue Dashboard
Interactive dashboard tracking revenue trends and top-performing products.
###  Key Features
* **Total Revenue KPI:** Real-time calculation of overall sales volume.
* **Product Performance Bar Chart:** Visual ranking of top-selling items like Laptops and Smartphones.
* **Chronological Trend Line:** Cross-filtered timeline showing monthly revenue peaks and valleys.
